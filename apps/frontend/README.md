# Frontend App

This directory is the CRM frontend only.

## Recommended Workflow

Run the stack from the repo root:

1. `npm run check`
2. `npm run dev`
3. `npm run stop`

## Run Locally

**Prerequisites:** Node.js and the backend app.


1. Install dependencies:
   `npm install`
2. Start the backend from `apps/backend`
3. Run the frontend:
   `npm run dev`
